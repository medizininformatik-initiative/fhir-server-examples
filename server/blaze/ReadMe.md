# Blaze #

## Start (manually) ##

`docker-compose -up -d`

`docker-compose logs -f`

## Configuration ##
Optional, edit `applications.yaml`

## Makefile (convenience) ##

`make start` 

`make stop`

`make clean`

`make bash`


## FHIR base ##

[http://localhost:8084/fhir](http://localhost:8084/fhir)

## Tested environment ##

  * Windows 10
  * wsl2
  * DockerDesktop
  * ubuntu
